## 📊 Sales Performance Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Sales Performance Dashboard built using Power BI.
The dashboard analyzes sales data from 2014–2017 and provides insights into sales trends, profit performance, regional performance, and product category analysis.
The goal of this project is to help businesses track KPIs, identify trends, and support data-driven decision making.

## 🎯 Business Objectives
Analyze sales performance over multiple years.
Identify best and worst performing regions and months.
Understand category and sub-category sales contribution.
Track sales growth and yearly performance.
Provide an interactive dashboard for business stakeholders.

## 📂 Dataset
The dataset used is the Sample Superstore dataset, which includes:
Order Date
Region
Category & Sub-Category
Sales
Profit
Quantity

2️⃣ Sales Analysis

3️⃣ Regional Analysis

4️⃣ Product Analysis
Sales by Sub-Category

5️⃣ Interactivity
Category slicer
Region slicer
Tooltip page
Navigation buttons
Dynamic filtering across visuals

## 📊 Key Insights
The West region recorded the highest sales.
July showed the lowest monthly sales.
Strong 69% sales growth observed across the analyzed period.
Technology category generated the highest profit.
Phones and Chairs are top selling sub-categories.
